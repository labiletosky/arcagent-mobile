ArcAgent

// autonomous on-chain commerce protocol

ArcAgent is a commerce and payments app on Arc. Users place orders, while the agent handles execution. It now uses USDC for testing and includes a direct faucet link to make the flow easier for new users.

Place Orders
Users can create simple on-chain orders with an item and amount.
Agent Execution
The approved agent wallet handles execution after the order is placed.
Track Everything
Lookup orders, copy order IDs, and follow the flow directly from the app.
How it works
01
Connect Wallet
Connect your Rabby or MetaMask wallet and switch to Arc Testnet.
02
Place Order
Enter an item name and USDC amount. Payment locks into the smart contract instantly.
03
Agent Executes
The approved agent confirms delivery and triggers payment release on-chain.
04
Payment Settles
Funds move instantly to the seller. No bank, no waiting, no middleman.
